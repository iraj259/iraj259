<h2 align="center">💟 src/pages/Projects.tsx 💟</h2>

> Real projects shipped in production or built to production-grade standards.

### 🤖 AI Automation Platform — Multi-Agent System
**LangGraph · RAG · Ollama · LangFuse · TypeScript**

Supervisor-based multi-agent architecture for task orchestration (customer support, calendar, content generation). RAG pipeline with document ingestion, chunking, Pinecone vector search, and local LLM support via Ollama. Full observability with LangFuse tracing.

---

### ⚡ LeetHub — Distributed Online Judge
**Node.js · Express · Fastify · React · MongoDB · Docker · Redis · Socket.IO**

Distributed online judge with Docker-based sandboxing for Python, Java, and C++ in isolated environments. Async job pipeline via Redis queues handling 100+ submissions/min; real-time feedback via Socket.IO. Stress-tested with k6.

---

### 📊 BetterStack Clone — Production Monitoring Platform
**Next.js · Node.js · PostgreSQL · Redis**

Multi-region health checks, real-time incident reporting, sub-second dashboard latency. Redis worker nodes reduced false-positive alerts by 30%. Automated incident lifecycle with email notifications and heartbeat monitoring for 99.9% uptime visibility.

---

### 🧠 AI Document Agent (@ Fundraisr)
**NestJS · TypeScript · LangChain**

MVP AI-powered document editor for surgical, format-preserving edits in DOCX/XLSX files. Structured ops pipeline with one-click rollback. Shipped into production.

---

### 🛫 Flight Booking System — Microservices Architecture
**Node.js · MySQL · MongoDB · Docker**

API Gateway with independent data stores per service, Docker Compose orchestration. Reduced environment setup time by 80%.
